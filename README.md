# S-derinswebbsida
En webbsida för Söderins där man kan hitta Söderins telefonnummer, en sida för att skicka ett meddelande och en för att göra en egen burgare, en sida om historian av Söderins, och en sida för menyn av Söderins.
För att gå in i projektet behöver du :
1.Ladda ner VS code.
2.Ladda ner tillägget Live Server eller Five Server.
3.När du laddat ner det så kommer det upp en knapp i hörnet av VS code som det står "Go Live" på.
4.När du trycker på "Go Live" så kommer det upp att VS code vill öppna webbsidan i webbläsaren.
5.Tryck på "Open in Browser" så kommer den öppnas i din default browser.
